# 問題概要

## 難易度

Normal

## 問題内容

整数の配列 nums と整数 k が与えられたとき、最も頻繁に出現する k 個の要素を返してください。答えはどのような順序であっても構いません。
答えは一意であることが保証されています。

Example 1:

> Input: nums = [1,1,1,2,2,3], k = 2
> Output: [1,2]

## 考えたこと

ハッシュマップに各値の出現回数を集計する
> {1:3, 2:2, 3:1}

その後頻度リストに格納する
> [0, 3, 2, 1]

最終的に逆順でループして必要な値だけ返す
> [1, 2]
